Aquatic Rush - Interactive web development game using WebGL.

Project Overview

Aquatic rush is a coin collecting game. The player will control the shark using the mouse. Players will have a total of 3 hearts. On the map there will be obstacles that the player must dodge while collecting coins that appear on the map.

Development Process

Aquatic rush is a pair project that I did with a friend in the course CS299., which is a Computer graphics course. It is a project developed with TreeJS on javscript based on simulating a scene on an html page.

First, I created a background scene with a loop object to simulate obstacles flowing towards the player. Represented by an orange square box. and create a green square box representing the player When the player moves the mouse left or right, the green box object moves as well.

After that I converted it by using the green box to simulate the player's hitbox and mocked up the game model so that it moves in relation to the hitbox and also added a simulation of generating additional coin hitboxes. You will notice that now there are two types of square boxes flowing towards the player.

When the player hits the coin hitbox, a coin will add to the player's score. And when the player collides with the hitbox that is an obstacle The player's health will decrease by 1 when they run out of health. The game will end.

These are all models added to the project. All the models were made by Inkrak through Blender.
